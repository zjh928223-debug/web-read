Warm reading surface, precise tool chrome.

The reading pane should feel calm and paper-like, with a generous measure, quiet margins, and no unnecessary interface chrome competing with the transcript.

The active sentence uses a warm highlight with a precise accent edge, so synchronized playback remains visible without turning the page into a dashboard.

Annotations should be present but low-noise. Counts, marks, and generated context belong near the text only when they help the reader recover meaning quickly.
